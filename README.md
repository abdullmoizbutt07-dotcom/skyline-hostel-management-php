# Skyline Hostel Management System

## Overview

Skyline Hostel Management System is a web-based application developed using Core PHP and MySQL to simplify hostel management operations. The system provides an efficient platform for managing students, room allocations, hostel records, and administrative activities through a user-friendly interface.

## Features

* Student Registration and Management
* Room Allocation and Management
* Hostel Record Management
* Admin Dashboard
* User Authentication
* Responsive User Interface

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* Bootstrap

## Installation

1. Download or clone the repository.
2. Place the project folder inside the XAMPP `htdocs` directory.
3. Start Apache and MySQL from XAMPP.
4. Import the provided SQL database into phpMyAdmin.
5. Open the project in your browser.
